Steps to setup the project - 

1. Install node.js, mongoDB, mongoDB compass and IDE (eg. VS Code).

2. Text editor like notepad is required to create csv files (delimiter is comma ‘ , ’).

3. Add nodejs path to environment variables

4. Install the node modules/ dependencies as mentioned in package.json using node package manager (npm). 
Command - npm i <dependency_name>

5. Provide the environment variables - MONGODB_URI,APP_PORT

6. Start the server using the following command - node server.js

7. Application will be accessible on following URL - http://localhost:3000/
